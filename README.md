# ADPulse

Analysis of Windows EVTX logs for detecting Active Directory attacks using YAML rules.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py --evtx-path ./evtx --rules-path ./rules --verbose
```
